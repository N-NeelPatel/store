## E-commerce store
This web app is very similar to Amazon. 

### Features
- Register or login as an user
- Update user account
- Search products
- Apply filters for searching products
- Add products to cart
- Change the required number for products
- Payment with PayPal
- Create order
- Check order history

### Packages used
- React-redux
- Material UI
- Mongoose
- Express
- Nodemon
